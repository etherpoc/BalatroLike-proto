# 『花札 × バラトロライク（仮）』総合仕様書 v1.0

## 1. プロジェクト概要
* **コンセプト:** 「花札（こいこい）」の役作りに、インフレするスコア計算とデッキ構築を融合。役が完成した際の「勝負（精算）」か「こいこい（超過プレイ）」かの究極のジレンマを軸にしたローグライク。
* **対戦形式:** 敵（AI）は存在しない完全ソロのスコアアタック。プレイヤーは限られた山札（デッキ）の中で、目標スコアの達成を目指す。
* **デザイン指針:** 「ネオ・トラディショナル」。黒背景に、極細のネオンラインで描かれた「月」「桜」「猪」などの抽象化されたアイコンのみで構成。和風の要素を削ぎ落とし、サイバーパンクのハッキング画面のような静かでストイックなビジュアル。

## 2. ゲームサイクルと「こいこい」システム
バラトロの「ディスカード（手札交換）」と「プレイ（提出）」の概念を、花札のルールに落とし込みます。

1. **ブラインド開始:** 目標スコアが提示される。山札（初期48枚）から、手札8枚・場札8枚が展開される。
2. **プレイ（手番）:** 花札の基本ルールに従い、手札から1枚出し、場札の「同じ月（植物）」のカードと合わせて獲得（札合わせ）する。その後、山札から1枚めくり、同様に場札と合わせる。
3. **役の判定:** 獲得した札の組み合わせが「役（猪鹿蝶、月見酒など）」を満たした瞬間、ゲームが一時停止し、究極の選択を迫られる。
    * **【勝負（SHOUBU）】:** 現在の役のスコアを計算・獲得して、獲得した札をリセットする。（手数を1消費）
    * **【こいこい（KOI-KOI）】:** スコアを確定させず、そのまま手番を続行する。次に別の役（または同種の役の追加）が成立した際、最終倍率が跳ね上がる。
4. **バースト（死）:** 「こいこい」を宣言したにも関わらず、手札が尽きるまでに次の役を作れなかった場合、**そのラウンドで獲得予定だったスコアは「0」になり没収**される。
5. **クリアと進行:** 目標スコアに到達すればクリア。ショップで札の強化やアーティファクトを購入し、次へ進む。

## 3. コアメカニクス

### 3.1 札の価値（チップ）
花札の4つのレアリティが、そのままベースチップになります。
* **カス（24枚）:** 1枚につき チップ `+1`
* **短冊（10枚）:** 1枚につき チップ `+5`
* **タネ（9枚）:** 1枚につき チップ `+10`
* **光札（5枚）:** 1枚につき チップ `+20`

### 3.2 スコア計算ロジック
「勝負」を宣言した瞬間に計算が走ります。
`獲得スコア = (獲得札の総チップ) × (役のベース倍率 + こいこいボーナス) × (アーティファクト乗算)`

* **役の倍率（例）:**
    * カス10枚: 倍率 `+1.0` (以降1枚ごとに+0.1)
    * 猪鹿蝶: 倍率 `+5.0`
    * 五光: 倍率 `+50.0`
* **こいこいボーナス:** 「こいこい」を1回宣言して次の役を作って勝負するたびに、最終的な倍率が `×2`、`×3` と乗算されていく。（ハイリスク・ハイリターン）

## 4. ローグライク要素（デッキとシナジー）

### 4.1 アーティファクト（パッシブ装備 / 最大5枠）
| カテゴリ | 名前 | 効果 |
| :--- | :--- | :--- |
| **チップ系** | **雑草魂（カス特化）** | 獲得した「カス札」1枚につき、チップが `+1` ではなく `+15` になる。 |
| **倍率系** | **風流人（酒特化）** | 「月見酒」「花見酒」の役が成立した時、ベース倍率が `+15.0` される。 |
| **欲張り系** | **狂気の沙汰** | 「こいこい」を宣言するたびに、最終倍率が `×2` ではなく `×3` になるが、手札からランダムな1枚が消滅する。 |
| **乗算系** | **猪突猛進** | 「猪」の札を獲得したターンに「勝負」を宣言すると、最終倍率が `×2`。 |
| **ルール変更** | **密造酒** | 山札の中に、すべての月とマッチできる万能札「ワイルドカード」を2枚混ぜる。 |

### 4.2 コンシューマブル（使い切り・イカサマ枠）
タロットカードの代わりに「花札の柄」を直接書き換えるイカサマアイテムです。

| 名前 | 効果 | UX |
| :--- | :--- | :--- |
| **昇格の筆** | 手札または場札にある「カス札」を1枚選択し、「光札」に書き換える。 | アイコンが光り輝き、形が変化する。 |
| **季節はずれ** | 選択した札の「月（属性）」を、指定した別の月に強制変更する。 | |
| **強制収穫** | 場札にある任意の1枚を、手札を使わずに直接獲得（回収）する。 | |

### 5.1 UX・操作のポイント
* **マッチアシスト:** 手札のカードにカーソルを合わせると、場札にある「同じ月」のカードがネオンカラーで強調表示される。
* **こいこいの演出:** 「こいこい」を選択すると、画面全体が一瞬赤くフラッシュし、BGMのビートが早くなるなど、後戻りできない緊張感を演出する。
