# 『さめがめ × バラトロライク（仮）』総合仕様書 v1.0

## 1. プロジェクト概要
* **コンセプト:** 隣接する同色ブロックを消去する「さめがめ」に、インフレするスコア計算とデッキ構築を融合。一撃で消すブロック数が多いほど、倍率が指数関数的に跳ね上がる「溜め撃ち」の戦略性を重視。
* **デザイン指針:** 「プリズム・ミニマリズム」。漆黒の背景に、発光する幾何学図形（○、△、□、×、◇）のみ。消去時のフラッシュと、弾けるように飛び出す数字の演出に特化。
* **開発環境:** ブラウザ（HTML5 / CSS3 / Vanilla JavaScript）。DOMベースの実装。

## 2. ゲームサイクルとリフィル（補充）システム
さめがめは盤面が空になると終わるゲームですが、本作では「リフィル」をリソースとして管理します。

1. **ブラインド開始:** 目標スコアが提示される。
2. **パズルフェーズ:** 2つ以上隣接する同色ブロックをクリックして消去（Play）。
    * **重力と整列:** ブロックを消すと上のブロックが落下し、列が空になると左に詰まる。
3. **リフィル（補充）:** 盤面をこれ以上消したくない、または消せなくなった際、**「REFILL」ボタン**を押して新しいブロックを全補充する（「手数」を1消費）。
4. **目標到達と精算:** 目標達成で「ROUND CLEAR」ボタン出現。リバーシ同様、オーバーラン可能。
5. **次ラウンドへ:** 盤面の状態を引き継いだまま、次の目標へ。

## 3. コアメカニクス

### 3.1 盤面と操作
* **グリッド:** 12 × 10マス。
* **ブロック:** 5色の幾何学図形。
* **操作:** 左クリック。2つ以上繋がっているブロックを1回消すごとに「1プレイ（1手）」とみなす。

### 3.2 スコア計算ロジック（指数インフレ）
さめがめの醍醐味である「大量消し」を、計算式で強力にバックアップします。
`獲得スコア = (消した数 × 10 + チップ加算) × (ベース倍率 + 消去数ボーナス + 倍率加算) × (アーティファクト乗算)`

* **チップ:** 消した枚数1枚につき `10`。
* **消去数ボーナス（Mult）:** 消した枚数に応じて倍率が加算される。
    * 2〜4枚: `+0`
    * 5〜9枚: `+2.0`
    * 10〜19枚: `+8.0`
    * 20枚以上: `+20.0`
* **全消し（フルクリア）ボーナス:** 盤面を1枚残らず消し去った場合、最終倍率が `×5.0`。

### 3.3 ゲームオーバー条件
* 目標スコア未達成の状態で「手数（Play）」が0になり、かつ盤面に消せるペアが存在しない場合。

## 4. ローグライク要素

### 4.1 アーティファクト（パッシブ装備 / 最大5枠）
「特定の色」や「消し方」に特化したシナジーを構築します。

| カテゴリ | 名前 | 効果 |
| :--- | :--- | :--- |
| **色特化系** | **情熱の赤（△）** | 赤いブロックを消した時のみ、ベース倍率に `+5.0`。 |
| **形状系** | **スクエア・ルート** | 縦横が同じ長さの「正方形に近い形」で消去された時、最終倍率を `×2`。 |
| **コンボ系** | **チェイン・リアクション** | 落下後に自動で連鎖（勝手に消去）が起きた場合、その連鎖の倍率を `×3`。 |
| **延命系** | **リサイクル・プラント** | 一度に15枚以上消去した時、20%の確率で「手数」を `+1` 回復。 |
| **経済系** | **プリズム・マイニング** | 5色すべてを1ラウンド中に1回ずつ消すと、クリア資金 `$10`。 |

### 4.2 コンシューマブル（使い切りアイテム / 最大2枠）
盤面の色をコントロールし、巨大な塊を作るための必須アイテム。

| 名前 | 効果 | UX |
| :--- | :--- | :--- |
| **ペンキ塗り** | 盤面の3x3の範囲を選択し、指定した1色に強制的に塗り替える。 | 範囲がスプレーを吹きかけたように一瞬で染まる。 |
| **解体ハンマー** | 隣接ペアのない「孤立した1つのブロック」を選択して消去する。 | 邪魔な1枚を砕き、巨大な塊同士を連結させる。 |
| **重力操作** | 次の1回のみ、重力の方向を「左」または「右」に変更する。 | 盤面全体が90度傾くようなグリッチ演出。 |

## 6. JavaScript実装ロードマップ

* **Phase 1: さめがめコアロジック**
    * 12x10の二次元配列生成。
    * 再帰アルゴリズムによる隣接ブロックの探索と削除。
    * 落下（Gravity）と列詰め（Reflow）の実装。
* **Phase 2: 補充とラウンド管理**
    * 「REFILL」ボタンの実装と、新しいブロックのランダム生成。
    * 目標スコア判定と、オーバーラン/精算システムの構築。
* **Phase 3: スコア計算と視覚演出**
    * 消去枚数に応じた指数的な倍率加算の実装。
    * 消去された場所から数字が飛び出し、左ペインの合計スコアに吸い込まれる演出。
* **Phase 4: アーティファクトのフック実装**
    * 「特定の色を消したか」「何枚消したか」をトリガーに、計算式に介入するシステムの構築。
