# 『タイピング × バラトロライク（仮）』総合仕様書 v1.0

## 1. プロジェクト概要
* **コンセプト:** 「QWERTYキーボード」の26文字がプレイヤーのデッキ。キー（文字）自体を強化し、画面に提示された単語のリストから「自分の強化したキーが多く含まれる単語」を選んでタイピングすることで、スコアを爆発させるローグライク。
* **戦略の核:** 「タイピング速度・正確性」というプレイヤースキルと、「どの文字を強化するか」というビルド構築の掛け合わせ。
* **デザイン指針:** 「ターミナル・ストイック」。真っ黒な背景に、等幅フォント（Monospace）のテキストと点滅するカーソル（`_`）のみ。エフェクトは文字の明滅や、コマンドが実行されたようなテキストのスクロールのみで表現する。

## 2. ゲームサイクルと「単語選択」
通常の手札の代わりに、画面に「お題のプール」が表示されます。

1. **ブラインド開始:** 目標スコアが提示される。画面にランダムな英単語（または短いコードスニペット）が「10個」リストアップされる。
2. **ターゲット選択（Play）:** プレイヤーはリストの中から好きな単語を選び、その最初の文字をタイピングすることでターゲットをロックする。
3. **タイピングフェーズ:** ロックした単語を最後までタイピングし終えると「1手（Hand）」を消費し、スコアが計算される。
4. **ミス（Typo）のペナルティ:** タイピングをミスすると、その単語での「コンボ倍率」が途切れるか、最悪の場合「手数（Hand）」が減るなどのペナルティが発生する。
5. **クリアと進行:** 目標スコアに到達すればクリア。ショップで「特定のキー（文字）」を強化するアイテムを購入し、次ラウンドへ。

## 3. コアメカニクス

### 3.1 キーボードデッキと文字の価値（チップ）
A〜Zの各キーには、出現頻度や打ちにくさに応じた「基本チップ」が設定されています。（スクラブルのようなイメージです）
* **母音（A, E, I, O, U）:** 出現しやすいが価値が低い（例: 1文字 `+2` チップ）
* **標準子音（S, T, N, Rなど）:** 標準的な価値（例: 1文字 `+5` チップ）
* **レア子音（Z, X, Q, Jなど）:** 打ちにくく出現しにくいが価値が高い（例: 1文字 `+20` チップ）

### 3.2 スコア計算ロジック
単語を打ち終えた（Enterを押した）瞬間に計算が走ります。
`獲得スコア = (単語の文字チップ合計 + 加算チップ) × (単語長のベース倍率 + キー固有の倍率 + WPMボーナス) × (アーティファクト乗算)`

* **単語長のベース倍率:** 3文字なら `×1.0`。1文字増えるごとに `+0.5` など、長い単語ほど倍率が高い。
* **WPM（入力速度）ボーナス:** その単語を打ち終えるまでのスピードが速いほど、最終倍率にボーナスが掛かる（スキル要素）。
* **キー固有の倍率:** ショップで強化した特定のキー（例：「S」のキー）を打った時だけ、倍率が加算される。

## 4. ローグライク要素（ビルド構築）

### 4.1 アーティファクト（パッシブ装備 / 最大5枠）
タイピングのスタイルや、特定のキー配列を強化するジョーカー枠。

| カテゴリ | 名前 | 効果 |
| :--- | :--- | :--- |
| **配列系** | **ホームポジション** | 「A, S, D, F, G, H, J, K, L」のキーを打つたびに、倍率が `+1.0` される。 |
| **技術系** | **メカニカル・スイッチ** | タイピングミス（Typo）を一度もせずに単語を完成させた場合、最終倍率が `×2`。 |
| **文字系** | **母音の反逆** | 「A, E, I, O, U」の基本チップが `+20` に跳ね上がる。 |
| **条件系** | **キャメルケース** | 大文字と小文字が混ざった単語をタイピングした時、倍率が `+5.0` される。 |
| **狂気系** | **ブラインドタッチ** | タイピング中、画面上の「次に入力すべき文字」が `*` で隠される代わりに、全スコアが `3倍` になる。 |

### 4.2 コンシューマブル（キーキャップ交換・タロット枠）
A〜Zの特定のキー自体に「修飾語」を付与して強化する、本作独自のハック要素です。

| 名前 | 効果 | UX |
| :--- | :--- | :--- |
| **金メッキのキートップ** | 指定したキー（例:「K」）を強化する。以降、その文字を打つたびにクリア資金が `$1` 増える。 | ターミナル上で該当文字が黄色くハイライトされる。 |
| **ガラスのキートップ** | 指定したキーを強化する。打つと倍率が `×2` になるが、30%の確率でその文字が一時的に「故障（入力不可）」になる。 |
| **正規表現（RegEx）** | リストにある単語をすべてリロード（再生成）する。手数は消費しない。 | `> reloading words...` と表示されリストが更新される。 |

完全なCUI（キャラクターユーザーインターフェース）を模したデザインにします。

```text
root@balatro:~# ./typing_game --target 250000
=========================================================
TARGET: 250,000      CURRENT: 84,200      HANDS: 3
ARTIFACTS: [HomePosition] [MechanicSwitch] [VowelRebel]

  5.  OVERFLOW       (8 chars)

  [チップ 45] × [倍率 8.5] (WPM: 120)
  + 382 点...
```

### 5.1 UX・操作のポイント
* プレイヤーがキーを叩くたびに、カタカタという小気味よいメカニカルキーボードの打鍵音（ASMR的な要素）を鳴らす。
* 単語を打ち終えてスコアが計算される際、ターミナルのログが高速で流れるような演出（`[OK] Chip added... [OK] Mult applied...`）を入れ、ハッカーになったような万能感を演出する。
