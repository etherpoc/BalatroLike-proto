# BalatroLike

パズルゲームに『バラトロ』のローグライク要素を融合させたブラウザゲーム集。
GitHub Pages でホスティングし、ポータルページから各プロトタイプにアクセスできます。

## プロトタイプを追加するには

1. ルートに新しいディレクトリを作成（例: `minesweeper/`）
2. そのディレクトリ内に `index.html` を配置
3. `index.html`（ポータル）の `GAMES` 配列にエントリを追加

## 2048 × バラトロライク

数字を合体させる『2048』のパズル性に、スコア倍率が掛け算でインフレする『バラトロ』の戦略性を融合させたローグライク。

- 操作: 矢印キー / WASD / タッチスワイプ

## リバーシ × バラトロライク

リバーシの石返しの快感をスコアインフレに変換するローグライク。黒石が増えるほど寿命が縮まるジレンマ。

- 操作: マウスクリック

## 数独 × バラトロライク

数独の配置ルールに手札マネジメントとライン消去を融合。列やブロックを揃えて倍率を爆発させるローグライク。

- 操作: マウスクリック（手札選択 → 盤面配置）

## さめがめ × バラトロライク

隣接する同色ブロックを消去するさめがめに、指数インフレするスコア計算を融合。溜めて一気に消すほど倍率が跳ね上がる。

- 操作: マウスクリック
