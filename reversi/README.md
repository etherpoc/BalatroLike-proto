# 『リバーシ × バラトロライク（仮）』総合仕様書 v4.0

## 1. プロジェクト概要
* **コンセプト:** 「リバーシ」の対戦ルールをベースに、石をひっくり返す快感を「スコアインフレ」に変換するデッキ構築ローグライク。
* **戦略性:** 目標達成後の「チキンレース（オーバーラン）」機能を搭載。「次ラウンドのために盤面を整えるか」「限界まで稼ぐか」の選択がプレイヤーの運命を左右する。
* **デザイン指針:** 「サイバー・ストイック」。黒背景に、ネオンフォントの数字と発光する石のみで構成されるミニマルなビジュアル。

## 2. ゲームサイクルとチキンレース進行
ラウンド間で**「石の配置を引き継ぐ」**シームレス進行をベースに、プレイヤーの任意によるラウンドクリアを採用します。

1. **ブラインド開始:** 目標スコアと、AIの思考レベルが提示される。
2. **プレイヤーターン（黒）:** 石を置き、白石をひっくり返してスコアを稼ぐ。
3. **目標到達と選択（オーバーラン）:** 合計スコアが目標に到達すると**「ROUND CLEAR」ボタン**が出現。
    * **選択A（精算）:** ボタンを押し、安全にショップフェーズへ移行する。
    * **選択B（続行）:** ボタンを押さずに黒石を置き続け、さらなるスコアや「オーバーラン専用アーティファクト」の成長ボーナスを稼ぐ。
4. **NPCターン（白）:** プレイヤーが石を置いた後、AIが思考ルーチンに基づき白石（的）を置く。
5. **次ラウンドへ:** 精算後、**盤面を引き継いだ状態**で次のラウンド（より高い目標・より強いAI）へ進む。

## 3. コアメカニクス

### 3.1 終了・ペナルティ条件（重要）
盤面が埋まる、または白黒両者ともに打てるマスがなくなり「対局が終了（詰み）」したタイミングによって、結果が変わります。

* **【完全な死（Run終了）】:** **目標スコアに到達する前**に盤面が詰まった場合。即座にゲームオーバーとなる。
* **【欲張りの代償（強制進行ペナルティ）】:** **目標スコア到達後（オーバーラン中）**に盤面が詰まった場合。
    * **ペナルティ内容:** 目標スコアを超過して稼いだ「余剰スコア」と、それに紐づく「ボーナス資金」をすべて没収される。
    * **強制進行:** ゲームオーバーにはならないが、ショップフェーズを経由した後、**「打てる場所がほぼない（詰みかけの）盤面」のまま、強制的に次のラウンドへ進まされる**という重い十字架を背負うことになる。

### 3.2 敵AIのレベルデザイン
* **Lv 1（序盤）:** ランダム・強欲配置。
* **Lv 2（中盤）:** 警戒配置（角周りを避ける）。
* **Lv 3（終盤）:** モビリティ重視（プレイヤーの選択肢を奪う）。

### 3.3 スコア計算ロジック
`獲得スコア = (ひっくり返した枚数×10 + チップ加算) × (ベース倍率 + 役ボーナス + 倍率加算) × (アーティファクト乗算)`

* **役ボーナス:** 角（四隅）に置くと倍率 `+3.0`。辺（外周）に置くと倍率 `+1.0`。

## 4. ローグライク要素（オーバーラン対応版）

### 4.1 アーティファクト（最大5枠）
| カテゴリ | 名前 | 効果 |
| :--- | :--- | :--- |
| **チップ系** | **辺の守護者** | ひっくり返した石の中に「辺」が含まれていた場合、1枚につきチップ `+30`。 |
| **倍率系** | **角の支配者** | 「角」を取った時の役ボーナスが `+10.0` に強化される。 |
| **欲張り系** | **強欲の壺** | オーバーラン中に稼いだスコア「10,000点」につき、クリア時の獲得資金 `$1` 増加。 （※ペナルティを受けると無効になる） |
| **欲張り系** | **コレクター** | オーバーラン中に白石を1枚ひっくり返すたび、ベースチップが永続的に `+1`。 |
| **経済系** | **黄金の楔** | 「角」に石を置いた状態でラウンドをクリア（精算）すると、ボーナス資金 `$15`。 |

### 4.2 コンシューマブル（最大2枠）
| 名前 | 効果 | UX |
| :--- | :--- | :--- |
| **ホワイトアウト** | 盤面にある任意の自分の「黒石」を最大5つまで「白石」に戻す。 (※次ラウンドに向けて盤面を軽くする必須級アイテム) | 対象の石が白く反転。 |
| **地殻変動** | 盤面のすべての石を、白黒反転させる。 | 画面が一瞬暗転し、ネオン発光。 |
| **強行突破** | 次の1手のみ、白石を1枚もひっくり返せない「空きマス」にも黒石を置ける。 | 全空きマスが発光。 |
