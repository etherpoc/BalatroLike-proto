# 『2048 × バラトロライク（仮）』総合プロトタイプ仕様書 v4.0

## 1. プロジェクト概要
* **コンセプト:** 「盤面が詰まる恐怖（寿命）」と戦う『2048』のパズル性に、計算式が掛け算でインフレする『バラトロ』の構築要素を融合させたローグライク。
* **ターゲット動作環境:** PCブラウザ（Google Chrome等） / キーボード操作前提
* **想定技術スタック:** HTML5 / CSS3 / JavaScript（Vanilla JS推奨。DOM要素で構築）
* **デザイン指針:** 装飾を極限まで削ぎ落とし、数字の合体とスコア計算のポップアップ（気持ちよさ）にフォーカスしたミニマルUI。

## 2. ゲームサイクル（シームレス進行）
「手数（制限回数）」を廃止し、**「盤面状況を次のラウンドに引き継ぐ」**ことで、途切れない生存競争を生み出します。

1. **ブラインド開始:** 現在の目標スコア（例：5,000点）と、特殊ルール（ボスブラインド時）が提示される。
2. **パズルフェーズ:** プレイヤーは無制限にスライド可能。合体するたびにスコアが加算される。
3. **ラウンドクリア:** 盤面が詰まる前に合計スコアが目標に到達した**瞬間にクリア**となり、パズルが一時停止する。
4. **ショップフェーズ:** スコアに応じた資金を獲得し、アーティファクト（パッシブ）やコンシューマブル（使い切り）を購入する。
5. **次ラウンドへ:** **盤面のタイル配置をそのまま引き継いだ状態**で、次の目標スコアに向けて再開する。

## 3. コアメカニクス

### 3.1 盤面と操作・ゲームオーバー条件
* **グリッド:** 4 × 4マス。
* **操作:** 矢印キー（上下左右）または WASDキー。
* **タイル生成:** 1スライド（タイルが動いた場合のみ）につき、空きマスに「2（確率90%）」または「4（確率10%）」がランダム出現。
* **ゲームオーバー（唯一の死）:** 盤面が完全に埋まり、**上下左右どこにスライドしてもタイルが合体できない状態**になること。

### 3.2 コンボシステム
スライドによって「合体（成長）」が連続して発生した際、プレイヤーにボーナスを付与します。
* **コンボの加算:** タイルが1組以上合体したスライドを「成功」とし、コンボ数を `+1` します。
* **コンボのリセット:** タイルが1組も合体しなかったスライド（空振り）が発生した瞬間、コンボ数は `0` にリセットされます。
* **コンボボーナス:** スコア計算時、現在のコンボ数に応じた倍率加算が発生します。
    * *標準計算式:* `獲得スコア = (チップ) × (倍率 + (コンボ数 × 0.2))`
    * ※この数値はアーティファクトによって強化可能です。

### 3.3 スコア計算ロジック
`獲得スコア = (合体合計 + チップ加算) × (ベース倍率 + コンボボーナス + 倍率加算) × (アーティファクト乗算)`

* **チップ（基本点）:** そのスライドで新しく生成された数字の合計。
* **倍率（Mult）:** 初期値は `×1`。
* ※何も合体しなかったスライドでは、スコア計算は発生しない。

## 4. ローグライク拡張要素（システム定義）

### 4.1 アーティファクト（パッシブ装備 / 最大5枠）
スコア計算式に介入し、インフレを引き起こすメイン要素。

### 4.2 コンシューマブル（使い切りアイテム / 最大2枠）
盤面の寿命（詰み）を直接的に解消、または一時的な超絶バフを与える救済措置。

### 4.3 特殊タイル（盤面ギミック）
* **ガラスタイル:** 合体時に倍率が2倍になるが、30%の確率で砕け散って消滅する。
* **黄金タイル:** 盤面に残したままラウンドクリアするとボーナス資金を獲得。

### 4.4 ボスブラインド（縛りルール）
* **重力異常:** 上方向にスライド不可
* **沈黙:** 「4」の合体ではスコアが入らない
* **中央集権:** 外周での合体ではスコアが入らない

> アーティファクト・コンシューマブルの全リストは [artifacts_and_items.md](./artifacts_and_items.md) を参照。

## 6. JavaScript実装ロードマップ

* **Phase 1:** コアエンジン（盤面・スライド・合体・詰み判定）
* **Phase 2:** バラトロサイクル（チップ×倍率計算・目標スコア判定）
* **Phase 3:** コンボシステム（連続合体トラッキング・倍率加算・UIメーター）
* **Phase 4:** Hookシステム（アーティファクトが計算式に介入）
* **Phase 5:** ショップ・コンシューマブル・特殊タイル
* **Phase 6:** ボスブラインド
